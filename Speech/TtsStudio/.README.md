# TtsStudio

A lightweight **Text‑to‑Speech (TTS)** module built with **.NET 10 Web API** and **React + Vite + Tailwind CSS**.  
Provides a simple UI for entering text, selecting a voice, and generating audio using **Azure Speech Service**.

---

## Tech Stack

### Backend (.NET 10)
- ASP.NET Core Web API  
- Azure Speech SDK (`Microsoft.CognitiveServices.Speech`)  
- Minimal API or Controller endpoint (`POST /api/tts`)  
- CORS enabled for local development  

### Frontend (React)
- React 18 + TypeScript  
- Vite  
- Tailwind CSS  
- Fetch-based API client  
- Audio playback via `<audio>` element

## Backend Setup

### Required NuGet Packages

- 
- Microsoft.CognitiveServices.Speech  (Azure Speech TTS SDK)


### Configuration

Add your Azure Speech credentials in `appsettings.Development.json`:

```json
{
  "AzureSpeech": {
    "SubscriptionKey": "YOUR_KEY",
    "Region": "YOUR_REGION"
  }
}
```

### Run Backend

```shell
cd backend/TtsStudio.Api
dotnet restore
dotnet run
```

Default: http://localhost:5000

---

## Frontend Setup

### Install & Run

```sh
cd frontend/tts-studio-react
npm install
npm run dev
```

Default: http://localhost:5173


### Voice Options

Grouped by region:

- US English  
- UK English

## API Contract

### POST /api/tts

Request:

```json
{
  "text": "Hello 你好",
  "voiceKey": "zh-cn-xiaoxiao"
}
```

Response:

- `200 OK` — audio stream (`audio/mpeg`)  
- `400/500` — error message

## Deployment Recommendation

- **Frontend:** Azure Static Web Apps (SWA)  
- **Backend:** Azure App Service (.NET 10)  
- **Proxy:** SWA API proxy → App Service `/api/*`  

This provides a clean, modern, low‑latency architecture.
