# tts-basic

A minimal Azure Text-to-Speech example.  
Converts text into audio and saves it as `output.wav`.

## Setup
1. Copy `.env.example` to `.env`
2. Fill in your Azure Speech key and region

## Dependencies

This module requires the following npm packages:

- **dotenv** — Loads environment variables from a `.env` file.
- **microsoft-cognitiveservices-speech-sdk** — Official Azure Speech SDK used for text‑to‑speech synthesis.

- **tsx** — Fast ESM runner with auto‑reload for development (dev‑only)

## Run
npm install  
npm start
